# 🚀 EduTrack – Classroom & Student Management Portal

## 📌 Overview
EduTrack is a full-stack Java-based application designed to manage classroom batches and student records efficiently. It provides a centralized system for handling academic activities using REST APIs and a Java-based client.

---

## 🧠 Tech Stack
- **Backend:** Java, Spring Boot  
- **Database:** MongoDB (NoSQL)  
- **Frontend:** Java CLI Client  
- **Architecture:** REST API (Client-Server)

---

## ✨ Features

### 📚 Batch Management
- Create, update, delete, and view batches
- Store details like batch name, trainer, schedule, and duration

### 👨‍🎓 Student Management
- Add students to batches
- View students in a batch
- Maintain student details (name, contact, enrollment)

### 🔁 CRUD Operations
- Full Create, Read, Update, Delete functionality via REST APIs

### 🗄️ Database Integration
- MongoDB for scalable and flexible data storage

---

## 🏗️ Architecture
Client (Java CLI) → Spring Boot REST API → MongoDB
